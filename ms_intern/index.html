About Us

TravEasy is a cutting-edge travel itinerary planner developed using Microsoft Azure resources, designed to enhance your travel experience with intelligent features and seamless functionality. With TravEasy, users can effortlessly locate the nearest hotels, restaurants, and tourist spots using Azure Maps, ensuring they have all essential information at their fingertips. Additionally, TravEasy features a chatbot assistant powered by Azure Bot Service and Azure Cognitive Services, which provides personalized activity suggestions based on the user's current location and preferences. Whether you're planning a vacation or exploring a new city, TravEasy makes travel planning simple, intuitive, and tailored to your needs.
